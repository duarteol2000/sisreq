Bibliotecas Python de terceiros utilizadas no projeto:

- Django (framework web principal do projeto, usado em toda a camada backend).

Observação: além de Django, o código só importa módulos da biblioteca padrão do Python
(por exemplo: os, pathlib, json, collections), que não precisam ser instalados via pip.

Bibliotecas JavaScript/CSS utilizadas via CDN:

- Bootstrap 5.3.0 (CSS e JS), carregado via:
  https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css
  https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js
- Chart.js (versão fornecida pela URL genérica do CDN), carregado via:
  https://cdn.jsdelivr.net/npm/chart.js

Essas bibliotecas front-end são usadas nas templates HTML em:
- apps/templates/core/base.html
- apps/templates/core/dashboard.html
- apps/estoque/templates/estoque/relatorio_entrega_material.html
- apps/estoque/templates/estoque/relatorio_consumo_usuario.html
- apps/estoque/templates/estoque/relatorio_consumo_setor.html
- apps/estoque/templates/estoque/relatorio_requisicoes_periodo.html
